# Game table background placeholder

The game now looks for this file first:

```text
assets/background/game_table.png
```

To replace the table during matches, put your final table artwork here with the same filename.
Recommended size: 1920x1080 PNG or JPG, 16:9.

The code scales and crops the image automatically to the selected resolution. If `game_table.png` is missing or broken, the old procedural table background is used as a fallback.
