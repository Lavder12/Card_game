# Патч: фон главного меню

Что изменено:

1. В `assets/background/` добавлен стабильный файл `main_menu.png` на основе уже имеющегося фонового изображения.
2. В `main.py` добавлены функции:
   - `load_menu_background()`
   - `_scale_menu_background_to_screen()`
   - `draw_main_menu_background()`
3. В `main_menu()` вместо обычного карточного стола теперь рисуется фон из `assets/background/main_menu.png`.
4. При смене разрешения фон автоматически масштабируется под размер окна.
5. Остальные экраны игры продолжают использовать старый фон карточного стола.

Если нужно заменить картинку позже — просто замени файл:

```text
assets/background/main_menu.png
```

Главное: оставить это же имя файла.
