Targeting and Card Size Patch
=============================

Changes:
- fixed route score badges so bot/player scores no longer overlap in the middle;
- player scores now anchor above the player's actual card stack;
- bot scores now anchor below the bot's actual card stack;
- regular number cards can now be played by clicking the whole player route area;
- face cards (J, Q, K, Joker) still require an exact target number card;
- increased card size on the table and in hand;
- increased hand area slightly so larger cards fit more comfortably.
