# Пыльный тракт — сюжетные этапы

Основной файл кампании: `stages.json`.

Игра загружает этапы из этого файла. Если файл удалить или сломать, `main.py` использует встроенный fallback, чтобы проект не падал.

Главное правило кампании: следующий этап открывается только после победы в партии. При поражении ключевая правда не выдаётся.

Флаги побед:
- win_juk
- win_mira
- win_torm
- win_nika
- win_velt
- win_orren
- win_roven
- win_grin
- win_klyk
- elie_saved
- truth_full
